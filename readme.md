# Helicopter simulator

This is our final project for Computer graphics 2021-2022, ZJU.



**Hot keys** of the project:

- N/M: swicth the view pattern (FPS / TPS)
- W: Move forward
- S: Move backward
- A: Move to left
- D: Move to right
- E: Open the door
- P: Screen shot (store in the "screenShot" file)
